# My Awesome PhoneBook

## 概要
このプログラムは C++98 で実装された簡易電話帳アプリケーションです。  
80年代の「しょぼいけど味のある」電話帳を模しており、以下の3つのコマンドを受け付けます:

- **ADD** : 新しい連絡先を登録
- **SEARCH** : 登録済みの連絡先を一覧・詳細表示
- **EXIT** : プログラムを終了

最大で **8件** の連絡先を保持し、9件目を登録すると最古のデータが上書きされます。  
動的メモリ確保は禁止されており、固定長配列で管理しています。


# COPY START（コマンド操作）

```markdown
### コマンド操作

1. `ADD`  
   - 5つの項目を順に入力します:  
     - First Name  
     - Last Name  
     - Nickname  
     - Phone Number  
     - Darkest Secret  
   - ルール:  
     - **空欄は禁止**（スペースやタブだけの入力も不可）  
     - **Phone Number は数字のみ**

2. `SEARCH`  
   - 登録済みの連絡先を一覧表示します。  
   - 一覧は **Index | First Name | Last Name | Nickname** の4列で、各列は幅10文字。  
   - 10文字を超える場合は **先頭9文字 + '.'** に省略されます。  
   - Index を入力すると、対応する連絡先の詳細（5項目すべて）がフルで表示されます。  

3. `EXIT`  
   - プログラムを終了します。  


## 仕様のポイント
- **最大8件**まで保存。9件目以降は**古いものから上書き**。  
- 一覧の表示は**右寄せ・10幅・省略表記**で整形。  
- 入力はすべて `std::getline` を使用。空白を含む入力にも対応。  
- データの妥当性チェックは `Contact` クラス内で一括保証。

## 学習の狙い

この課題を通して、C++ の基礎を実践的に学ぶことができます。

- **クラス設計**  
  - `Contact` クラスで「1件の連絡先」を表現  
  - `PhoneBook` クラスで「複数の連絡先」を管理  
  - クラスごとの役割分担を理解できる  

- **アクセス修飾子の理解**  
  - メンバ変数は `private` にして外部から直接触れないようにする  
  - 必要な操作だけを `public` 関数として公開する  
  - → **カプセル化** の基本を学ぶ  

- **入力検証と不変条件**  
  - 空欄や空白だけを禁止  
  - Phone Number は数字のみ  
  - データの妥当性を **クラス内部で保証** する設計を体験できる  

- **入出力と整形**  
  - `std::getline` による安全な文字列入力  
  - `std::setw` や独自関数 `fit10` による列整形  
  - 入力と出力を組み合わせて UI を作る練習  

- **C++ の基本構文**  
  - コンストラクタの利用  
  - `const std::string&` の参照渡し  
  - 固定長配列での管理（動的メモリ禁止の縛りを活かす）  

このようにして、「クラスの使い方」「カプセル化」「データ検証」「入出力整形」といった  
C++ 初学者に必須の知識を一通り学べるようになっています。

## ファイル構成と役割

### main.cpp
- エントリーポイント（`main` 関数）
- ユーザーからのコマンド入力を処理
- `ADD` / `SEARCH` / `EXIT` をハンドリング
- 実際の登録や表示処理は `PhoneBook` に委譲

---

### Contact.hpp / Contact.cpp
- **Contact クラス** を定義
- 1件の連絡先を表す
- メンバ変数（`first_`, `last_`, `nickname_`, `phone_`, `secret_`）は `private`
- `setField` で一括入力とバリデーション（空欄禁止・Phoneは数字のみ）
- `getXxx` 系のゲッタでデータを外部に公開
- `isEmpty` で有効かどうかを判定

---

### PhoneBook.hpp / PhoneBook.cpp
- **PhoneBook クラス** を定義
- 最大8件の `Contact` を固定配列で管理（動的メモリ禁止）
- `add(Contact)` : 新しい連絡先を追加（9件目で最古を上書き）
- `list()` : 一覧表示（10幅・右寄せ・長い文字列は `fit10` で省略）
- `showDetail(index)` : 指定したインデックスの詳細情報を表示
- 内部で `size_` と `next_` を持ち、リングバッファのように管理
