  - 署名は「誰かがしていればOK」ですが、実行者本人は実行等級を満たす必要があり
    ます。

  なので一般論としては、
  「自分は署名条件を満たさないけど、実行条件は満たす」ケースがあれば、他人が署
  名してくれていれば実行は可能です。

  ただしあなたのコードでは、各フォームの gradeToExecute が gradeToSign より厳し
  い（数値が小さい）ので、署名できない人は実行もできません。

  例:

  - clerk(140) は shrub に署名できる(145)が、実行できない(137)
  - intern(150) は署名も実行も不可
  - boss(1) は両方可能

  つまり「署名してもらえば誰でも実行できる」ではなく、実行できる人だけが実行で
  きる、が正確です。
