# C++ Module 08 summary

## ex00: Easy find
- 何をやるのか: テンプレート関数 `easyfind` を実装し、コンテナ内の整数を探索する（最初の一致を返す／見つからなければ例外やエラー値）。
- 学習目的: STLコンテナとアルゴリズム（`std::find` など）をテンプレートで扱う感覚を身につける。
- 必要な知識: テンプレート関数、STLコンテナ（vector, list など）、`<algorithm>` の基本、例外処理。

## ex01: Span
- 何をやるのか: 最大 N 個の整数を保持する `Span` クラスを作成し、`shortestSpan()` と `longestSpan()` を実装。範囲挿入も追加。
- 学習目的: コンテナ内部管理、アルゴリズム（ソート、差分計算）、例外設計、イテレータの活用。
- 必要な知識: STLコンテナ、`std::sort` 等のアルゴリズム、例外処理、イテレータ範囲挿入、計算量の意識。

## ex02: MutantStack
- 何をやるのか: `std::stack` を継承（またはラップ）し、イテレータで走査できる `MutantStack` を実装。
- 学習目的: アダプタコンテナの仕組み理解、継承とメンバ型の公開、イテレータ提供。
- 必要な知識: `std::stack` の内部コンテナ、継承、`typedef`/`using` でのイテレータ型公開、begin/end 実装。

## わからないワードまとめ
- コンテナ: 複数の要素をまとめて持てる入れ物。例: 配列、vector、list など。
- STLコンテナ: C++標準ライブラリにあるコンテナ群。例: `std::vector`, `std::list`, `std::deque`, `std::map` など。
- イテレータ範囲挿入: 「開始イテレータ〜終了イテレータ」でまとめて追加すること。例: `addNumber(first, last)` のように範囲で渡す。
- ラップ: 既存のクラスを「包んで」使うこと。中身を隠しつつ外側で独自の機能を追加するイメージ。
- アダプタコンテナ: 既存コンテナを元に機能を制限・変更したコンテナ。例: `std::stack` や `std::queue` は内部に `deque` などを使っている。
